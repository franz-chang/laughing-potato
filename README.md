# Distribution-Guided Generative Adversarial Attack on LLM

Project homepage built from the `franz-chang/DG_Attack` Docusaurus template style, customized for a focused 7-paper collection on adversarial attacks and evaluation in LLM/LVLM settings.

## Included Papers

1. Goal-guided Generative Prompt Injection Attack on Large Language Models
2. Target-driven Attack for Large Language Models
3. Semantic-Preserving Prompt Hijacking: A Black-Box Adversarial Attack on Auto-Prompt Optimization
4. Efficient and Stealthy Jailbreak Attacks via Adversarial Prompt Distillation from LLMs to SLMs
5. Self-Evolving Matrix-Poison Agent on Recommender System (local PDF included)
6. AttackEval: How to Evaluate the Effectiveness of Jailbreak Attacking on Large Language Models
7. Large Vision-Language Model Security: A Survey

## Setup

```bash
npm install
```

## Local Run

```bash
npm run start
```

## Build

```bash
npm run build
```

## Deployment Notes

- Update `url` and `organizationName` in [`docusaurus.config.js`](./docusaurus.config.js) before publishing.
- `baseUrl` is currently set for GitHub Pages project deployment:
  `/distribution-guided-generative-adversarial-attack-on-llm/`.
